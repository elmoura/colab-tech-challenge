## Descrição

> _Feito como desafio técnico para Colab S.A._

O projeto está separado entre API (NestJS) e front-end web (Vite + React), ambos escolhidos principalmente pra agregar agilidade na entrega, dada a rápida configuração dos 2. Nos 2 projetos foram aplicados conceitos de Clean Architecture a fim de manter as diferentes camadas do software desacopladas e de fácil alteração/incrementação.
